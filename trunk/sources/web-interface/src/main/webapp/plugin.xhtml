Visual Paradigm Plug-in

Since December 2011 we provide a plug-in for Visual Paradigm. The plug-in has the following basic functionalities:

  • Importing decisions from decisionrepository.com
  • Drawing a relationship view
  • Drawing a chronological view
  • Synchronize decisions to/from decisionrepository.com

Introduction

Installing the plug-in is quite simple. We provide an automatic installer which should work in most cases. However, should this not succeed you can always install it manually.

Automatic installation

Linux/OSX

Installing for Linux and OSX is made easy using a shell script, if it does not work, please use the manual installation guide (see the following section).

  • Download the following shell script from our download mirror http://dl.decisionrepository.com.
  • Make it executable

    chmod +x odr-plugin-installer.sh

  • Run it

    ./odr-plugin-installer.sh

If it does not work, please add an issue to our issue tracker. In order to resolve the problem, please state clearly which OS system you are using and the error you encountered.

Windows

For now please refer to the manual installation steps below.

Manual Installation

Use the manual installation if the automatic one fails or if you prefer to do it manually. Simply follow these steps:

  • Download the plug-in .zip package from http://dl.decisionrepository.com.
  • Find your Visual Paradigm installation, e.g., C:\Program Files\Visual Paradigm, /opt/VP_Suite, /usr/local/VP_Suite.
  • Make a directory called plugins in the Visual Paradigm folder.
  • Extract the zip package so that you have Visual Paradigm/plugins/ODR.
