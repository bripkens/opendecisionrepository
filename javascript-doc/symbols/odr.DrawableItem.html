Class odr.DrawableItem


Extends odr.Callback.
Base class for all entities that can be drawn. A class that inherits from this class states that it can be drawn and that it knows how to draw itself.
Defined in: drawing.entities.js.

{String} extendedId()
The extended id is the id which is used in the SVG. Besides the normal id which you can retrieve by calling odr.DrawableItem.id it also returns some meta information. Mostly this is an id prefix.
Returns:
{String} The extended id
id()

Always returns the automatically generated ID of this DrawableItem
paint(parent)
Draw the item to the canvas. Every implementation should call the odr.Callback.draw method afterwards.
Parameters:
{DOM element|jQuery element} parent Optional
An optional parent element. In the resulting SVG, the drawn item will be a child of the parent item. Can be used to group elements. If ommited, the SVG root element will be the parent element.
{DOM element|jQuery element|odr.DrawableItem} parent(parent)
Define a parent element for this DrawableItem. The parent element can act as a grouping element or container for it's child element. This value can temporarily be overriden by the parent parameter in odr.DrawableItem#paint. You can also retrieve the current value by calling this method without parameters.
Parameters:
{DOM element|jQuery element} parent Optional
The parent element
Returns:
{DOM element|jQuery element|odr.DrawableItem} The parent which was set or null if no parent was set. If you call this method with a parameter then the method will return the object on which you called the method (Fluent Interface)
repaint()
Redraw the item. Most items will use an optimized algorithm that only modifies the location of the item. Therefore this method should be called when dragging an item.
{Object|odr.DrawableItem} value(value)
You can attach an arbitrary value with each DrawableItem. This value may be everything but the most useful way to use it is to store the data which was used to generate this DrawableItem (like JSON). You can also retrieve the current value by calling this method without parameters.
Parameters:
{Object} value Optional
The value you want to attach or nothing

{Boolean|odr.DrawableItem} visible(visible)
Show or hide this DrawableItem. When changing the visibility of an iten it will be painted or removed, depending on the new visiblity. Initially each DrawableItem is visible. You can also retrieve the current value by calling this method without parameters.
