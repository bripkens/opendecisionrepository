<static> {odr.Rectangle} odr.addNode(name, stereotype, stereotypeToShow, x, y, visible, round, extraClasses)
Use this utility functionality to add nodes. It has several benefits as it will make sure that the node will be added to the nodes menu and it will also add the node to the status menu.
Defined in: drawing.menu.js.
Parameters:
{String} name
The name which should be shown for this node
{String} stereotype
The stereotype that will be visible in the status menu
{String} stereotypeToShow
The stereotype that will be shown above the name of the decision in the node and in the decisions menu
{Number} x
The x position
{Number} y
The y position
{Boolean} visible Optional
Whether this node is visible. Default = false
{Boolean} round Optional
Whether the node has round corners. Default = true
{String} extraClasses Optional
Some extra classes that you want to attach odr.Rectangle#extraClasses. Default = ""
Returns:
{odr.Rectangle} The new node. The node has been painted to the canvas!

<static> {Number} odr.scale(newScale)

Scale the association to a new level or retrieve the current scale level by calling the method without a parameter.

The default scale level is 1. The reduce the scale level by 10% set the scale level to 0.9.

Parameters:
{Number} newScale Optional
The new scale level
Returns:
{Number} Always the current scale level. When setting a new scale level, the new level is returned.
